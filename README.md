# 📊 Dashboard de Gestão Empresarial em Power BI

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de analisar o desempenho financeiro, comercial e operacional de uma empresa fictícia através do Power BI.

O dashboard foi construído utilizando modelagem dimensional, Power Query, DAX e visualizações interativas, permitindo acompanhar indicadores estratégicos e apoiar a tomada de decisão.

---

## 🎯 Objetivos

- Monitorar receitas, despesas e lucro.
- Avaliar o atingimento das metas financeiras.
- Analisar a evolução dos resultados ao longo do tempo.
- Identificar os principais clientes e produtos.
- Avaliar fornecedores e categorias de despesas.
- Gerar insights para suporte à tomada de decisão.

## 📈 Principais Insights

- A empresa atingiu **100,94% da meta anual**.
- Receita total de **R$ 18,05 milhões**.
- Lucro líquido de **R$ 4,78 milhões**.
- Margem de lucro de **26,5%**.
- O maior cliente representa apenas **6,96% do faturamento**.
- O maior fornecedor representa apenas **7,61% das despesas**.
- Dezembro foi o mês com melhor desempenho financeiro.
- Marketing foi a principal categoria de despesa.
